# 🛒 ShopEase - E-Commerce Website

## 📖 About

ShopEase is a beginner-friendly static e-commerce website built using HTML. This project was created to practice the fundamentals of web development, including creating multiple web pages, navigation, images, hyperlinks, forms, and organizing a website structure.

---

## ✨ Features

- Home Page
- About Page
- Contact Us Page
- Login Page
- Shopping Cart Page
- Notifications Page
- Navigation Menu
- Product Catalog
- Customer Service Section
- Company Information
- Social Media Links
- Payment Methods Section
- Newsletter Subscription

---

## 🛠️ Technologies Used

- HTML5
